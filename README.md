# CDR Group

**CDR Group** is an independent research-driven organization working at the intersection of  
**cognitive science, artificial intelligence, and computational systems**.

We focus on building **theoretical frameworks**, **agent architectures**, and **experimental systems** that explore how complex behavior, decision-making, and internal dynamics emerge from formal models.

## Core Philosophy

> If a phenomenon cannot be **modeled**, it cannot be **tested**.  
> If it cannot be tested, it does not belong to science.

CDR Group prioritizes:
- **Operational definitions**
- **Measurable internal states**
- **Reproducible experiments**
- **Minimal assumptions, maximal explanatory power**

We deliberately avoid speculative metaphysics and focus on **models that do real work**.
